# 2026-06-10 — 使い方ガイド(docs/guide.md)を追加

M3「美化系」が揃い、描き味の核(2 ブラシ・手ブレ補正・擬似筆圧)が固まったタイミングで、
ユーザー向けの **使い方ガイド `docs/guide.md`** を新設した。topic ブランチ `docs/guide`。

## 方針(別スレッドでの相談 → 確認済み)

- **性格**: 「操作の一覧」ではなく **やりたいこと別の実践マニュアル + 描き味のコツ**。
  README の「アプリ内の操作」は平坦な早見表なので重複させず、README からガイドへ誘導する。
- **深さ**: 配布も視野の網羅マニュアル(全機能カバー: 描画・色・コツ・レイヤー・アニメ・書き出し)。
- **図版**: `docs/images/` の既存デモ PNG を最大限流用し、不足分だけ**再現可能な `--demo-*`** で生成。
  手動 UI スクショは撮らない(再現性優先)。
- **言語/配置**: 日本語・単一ファイル(CLAUDE.md の言語規約)。

## 構成(やりたいこと別)

0 画面の見方 / 1 はじめての一筆 / 2 ブラシ使い分け / 3 色・濃さ・サイズ / 4 描き味のコツ(レシピ) /
5 レイヤー / 6 取り消し・保存・書き出し / 7 アニメーション / 8 ショートカット早見表 / 9 トラブルシュート。

操作子・ショートカットは実装(`CanvasView`/`AppDelegate`/`InspectorView`/`TimelineView`)と 1:1 に突き合わせて記述。

## 図版

既存流用: `m2-timeline-ui`(画面俯瞰)・`guide-watercolor`/`m3-sumi`(ブラシ)・`dwell-*`(ブルーム)・
`m3-stabilize-*`(手ブレ)・`m1-layers-*`(レイヤー)・`m1-undo-*`(取り消し)・`m2-spritesheet`/`m2-onion`。

新規(再現可能・`AppDelegate` に `--demo-guide` を追加して生成):
- `guide-hero.png` — トップの作例。水彩ウォッシュ(空 + 岸)→ 乾燥 → **墨レイヤーで葦のかすれ**を重ねる。
  滲み・色・レイヤー・かすれを 1 枚で見せる。スナップショットを目視で 1 回チューニング(葦が散漫 → 弓なりの数本に整理)。
- `guide-watercolor.png` — 水彩のみ段階(§1/§2 用のやわらかいストローク)。
- `guide-anim.gif` — 既存 `--demo-anim` の GIF をそのまま採用。

## ハマり / 判断

- **UI スクショの鮮度**: 流用できる UI 画像(`m1-ui`/`m2-timeline-ui`)はいずれも **M3 の「手ブレ」スライダ追加前**。
  再現可能なデモは素のキャンバス窓(インスペクタ/タイムラインが写らない)なので、フル UI の自動生成はできない。
  → `m2-timeline-ui`(最新で最も情報量が多い)を俯瞰図に使い、**手ブレスライダは本文で補足**して整合を取った。
  将来フル UI の撮り直しが要るなら手動キャプチャ(別途)。
- **作例のアート性**: スクリプト描画で「絵」を作るのは限界がある。葦 + ウォッシュという**機能が伝わる構図**に割り切った
  (画題の美しさより、滲み/かすれ/レイヤーが一目で分かることを優先)。
